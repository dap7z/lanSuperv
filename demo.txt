Cette application permet de scan son réseau local et d'afficher les machines connectées.
On peut également interagir via un système de plugins avec les autres PC sur lesquels on a également installé l'application.

Au final à partir d'une même interface web, vous pouvez notamment démarrer et éteindre un PC distant.
(requis pour détection : sur le PC distant, avoir autorisé le ping)
(requis pour démarrage : sur le PC distant, avoir activé la fonctionnalité WakeOnLan)
(requis pour extinction : sur le PC distant, avoir installé le logiciel lanSuperv)

_______________________________________________________________________________________________

Concernant les scans du réseau local, ils ne se font que :
- au démarrage de l'application
- lorsque la page web est affichée
- toutes les heures (paramétrable dans le fichier config.js)

Description du flux côté serveur :
1) quickScan : au chargement de la page (sur les PC déjà connus dans G.VISIBLE_COMPUTERS)
2) fullScan  : découvre tous les PC du réseau
2) onePcScan : dès qu'un PC est detecté (pendant le fullScan), ses statuts ping/http/socket sont vérifiés immédiatement.
3) dump du contenu de la base de données pour verifications eventuelles

Description du flux coté navigateur client :
1) gunJs.on() affiche tout changements et permet également le chargement initial.

Cela réduit le nombre de scans et affiche les PC dès leur découverte, sans attendre la fin du scan complet.
